Como disponibilizar para os alunos

Este jogo é um arquivo único HTML. Para os alunos abrirem em casa, hospede o arquivo e envie o link.

Opção mais simples: envie o arquivo HTML pelo AVA, Google Classroom, WhatsApp ou e-mail. O aluno baixa e abre no navegador.
Opção com link: renomeie o arquivo para index.html e publique em GitHub Pages, Netlify ou Vercel. Depois envie apenas o link.
Opção institucional: peça ao suporte da instituição para colocar o arquivo em uma pasta pública do site ou do AVA.

Observação: como é estático, não registra notas dos alunos. Ele serve como revisão autônoma.
